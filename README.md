# PCB Enclosure – atmega328p-ws2812b-controller
3D-printed enclosure for atmega328p-ws2812b-controller (https://github.com/aditya-1225/atmega328p-ws2812b-controller-pcb.git).

## Printing
Print:
- bottom_full.3mf
- top_full.3mf
- plunger.3mf

Bottom and top are printed as single pieces.

## Assembly
1. Place PCB into bottom enclosure
2. Install plunger
3. Fit top enclosure

## PCB Compatibility
- Board size: 41 × 50 mm
- Tested revision: v1.0
